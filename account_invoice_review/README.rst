=======================
Invoice Review Checkbox
=======================

Adds a simple checkbox "Reviewed" for supplier invoices.
This is to help accountant to keep track what has been already reviewed.

The checkbox is only informative, and has no functionality.

Installation
============

Install the module form Settings->Local Modules
